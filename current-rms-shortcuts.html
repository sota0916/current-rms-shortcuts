Previous（前へ）

前の期間に移動

画面の「前へ」ボタンをクリックします。カレンダーや日付ビューで前の日／週／月に戻る時に使います。

Shift + E
➡️

Next（次へ）

次の期間に移動

画面の「次へ」ボタンをクリックします。カレンダーや日付ビューで次の日／週／月に進む時に使います。

Shift + R
📅

Reset to Today（今日にリセット）

表示日を今日に戻す

表示中の日付を今日に戻します。Planner / Booking Availability / Product Availability / Calendar / Resource Planner に対応。

対応ページ: Planner, Availability, Calendar, Resource Planner

Shift + S
🗓️

Show Picker（日付ピッカーを開く）

日付選択ボックスを開く

画面上の「Show」ピッカーボタンを開きます。任意の日付を素早く指定したいときに便利です。

Shift + C
📝

Create Quotation（見積もり作成）

新規 Quotation を作成

画面右上の「Create」ドロップダウンを開き、「Quotation」を自動選択します。Opportunity 画面などから即座に見積を作成できます。

Shift + A
🏗️

Go to Job Planner（プランナーへ移動）

/planner ページに直接移動

どのページからでも一発で Job Planner（/planner）に移動します。

📌 使うときの注意

  • ショートカットは必ず Shift + 大文字キー の組み合わせです（Ctrl / Cmd / Alt は併用しません）。
  • テキスト入力中（input / textarea / 編集可能要素にフォーカス中）は 動作しません。先に Esc などでフォーカスを外してください。
  • ボタンが画面上に存在しない場合（例：Show ピッカーが無いページで Shift+S）は何も起きません。
  • 動作対象は https://*.current-rms.com/* のみです。

🚀 インストール手順

  1. Chrome に Tampermonkey 拡張をインストール（Chrome ウェブストア）。
  2. 上の 「⬇️ .user.js をダウンロード」 ボタンをクリック。
  3. Tampermonkey が自動でインストール画面を開くので 「Install」 を押す。
  4. Tampermonkey のダッシュボードで 有効 になっていることを確認。
  5. Current RMS を開いて、Shift + R などを試す。動けば導入完了！

※ うまく開かない場合は、GitHub リポジトリ から .user.js を直接コピーして、Tampermonkey ダッシュボードの「+」で新規スクリプトに貼り付けて保存してください。
